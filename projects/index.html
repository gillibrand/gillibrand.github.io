Global Configuration Management (GCM)

GCM manages versioned artifacts across apps in the IBM Engineering Lifecycle Management (ELM) suite. I wrote and maintained almost all of its web UI, many REST services, and some of the Java backend services.

Tree Component

I wrote GCM's main tree component to support lazy-loading, fast re-renders, animated changes, keyboard navigation, multi-select, drag-and-drop, and more.

Tree Compare

The above tree component is configurable and used in different places. It's used in this compare view with fewer actions enabled, but unique styling, and state and scroll syncing between the two.

Query Builder

The GCM query builder UI can search for streams based on attributes like dates, tags, numbers, custom enums, and more. It can nest boolean conditions, and use other saved queries as subqueries. It's easy to refine complicated queries by previewing results inline and using drag-and-drop to move conditions between groups.

Beside writing the UI, I wrote the Java services that handled its internal domain-specific language, and that translated that into the actual database queries.

History on Canvas

This history view shows the different branches and baselines of a stream. I used a <canvas> element to to draw the arrows. Scalable to thousands of entries. The optional mini-map in the bottom right can pan around large graphs.

DOORS Next

IBM DOORS Next is a market leading requirements management tool that I worked on for a short time. I contributed many bug fixes and serviceability improvements, but I also had the opportunity to refresh its query builder (aka, filters).

Carbon Query Builder

The query builder was an existing feature. I revised it to align with the IBM Carbon Design guidelines, added support for nested AND/OR groups, drag-and-drop reordering, and improved accessibility and usability.

Inline Delete

The old query builder design let you remove conditions with a single click. For the new design I added inline "Remove" confirmations directly on the buttons. My comprise that added a safety-net, but kept the speed of editing customers expected.

Pix Grid for Playdate

A pixel art app for Playdate. Written in Lua. Available on GitHub.

URL Clown for Chrome

A Chrome extension for editing URLs. I needed this for work and existing extensions didn't handle hash parameter key-value pairs. Available on Chrome Web Store and Github.
