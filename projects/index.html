Global Configuration Management (GCM)

GCM manages versioned artifacts across apps in the IBM Engineering Lifecycle Management (ELM) suite. I wrote and maintained almost all of its web UI, many REST services, and some of the Java backend services.

Tree Component

I wrote GCM's main tree component to support lazy-loading, fast re-renders, animated changes, keyboard navigation, multi-select, drag-and-drop, and more.

Query Builder

I wrote the GCM query builder UI, plus the Java services to parse its internal domain-specific language and translate that into the actual database queries. It can search based on various attribute types, nest boolean conditions, and be edited using drag-and-drop.

History on Canvas

This history view shows the different branches and baselines of a stream. I used <canvas> to to draw the connections. The optional mini-map in the bottom right can pan around large graphs. There are several ways to collapse, filter, and navigate to support thousands of loaded nodes.

DOORS Next

IBM DOORS Next is a market leading requirements management tool that I contributed many bug fixes and serviceability improvements to, and also had the opportunity to refresh its query builder (aka, filters).

Carbon Query Builder

I revised the DOORS Next query builder it to align with the IBM Carbon Design guidelines, added support for nested AND/OR groups, drag-and-drop reordering, and improved accessibility and usability.

Inline Delete

The old design let you remove conditions with a single click. That was too error-prone, so I added inline "Remove" confirmations as a safety-net that kept the speed of editing users expected.

Report Builder

Report Builder creates and runs reports across the IBM ELM suite. I bootstrapped the project to replace its jQuery UI with a modern TypeScript/React UI. That included initial project setup, integration with with existing backend services, and building some of its core UI components.

Bulk Tag Editor

I built a React tag component that needed a variation for when multiple reports are selected. I worked closely with a UX designer on this version that uses menus, icons, and color to clearly communicate all changes that will happen.

Pix Grid for Playdate

A pixel art app for Playdate. Written in Lua. Available on GitHub.

URL Clown for Chrome

A Chrome extension for editing URLs. I needed this for work and existing extensions didn't handle hash parameter key-value pairs. Available on Chrome Web Store and Github.
